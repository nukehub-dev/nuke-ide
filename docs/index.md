## What is NukeIDE?

NukeIDE is an open-source Integrated Development Environment (IDE) specifically designed for nuclear simulation and engineering. Built on the extensible Theia platform, it provides a robust and customizable environment for computational physics, reactor design, fusion energy, and radiation transport.

Whether you're running OpenMC Monte Carlo simulations, visualizing DAGMC geometry, or managing complex Python environments for scientific computing, NukeIDE brings everything together in one unified workspace.
