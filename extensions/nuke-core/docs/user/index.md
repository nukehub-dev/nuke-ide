# User Documentation

End-user guides for the Nuke Core extension, the foundational Python environment manager for NukeIDE.

## Guides

| Guide                                                   | What You'll Learn                                                                           |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| [**Getting Started**](getting-started.md)               | First-time setup, configure Python, run a health check, set OpenMC cross-sections           |
| [**Environment Management**](environment-management.md) | Auto-detection, switching, creating, and deleting environments; workspace auto-detect       |
| [**Package Installation**](package-installation.md)     | Install packages with pip, uv, or conda; custom channels and indexes; live terminal output  |
| [**Troubleshooting**](troubleshooting.md)               | Fixes for common issues: missing Python, failed installs, status bar, cross-sections errors |

## Quick Commands Reference

| Command                        | Purpose                                                       |
| ------------------------------ | ------------------------------------------------------------- |
| `Nuke: Switch Environment`     | Pick a different Python environment                           |
| `Nuke: Environment Actions`    | Switch, open terminal, install packages, copy path, or delete |
| `Nuke: Create Environment`     | Create a new conda or venv environment                        |
| `Nuke: Delete Environment`     | Remove a user-created environment (type-to-confirm)           |
| `Nuke: Install Package`        | Install packages with live terminal output                    |
| `Nuke: Run Health Check`       | Validate environment, tools, and optional packages            |
| `Nuke: Validate Configuration` | Quick check of settings and paths                             |
| `Nuke: Show Diagnostics`       | Full diagnostic report for troubleshooting                    |
