# NukeLab Menu Extension

JupyterHub integration menu for NukeIDE deployments.

## Overview

`nukelab-menu` adds navigation links for NukeLab-based deployments of NukeIDE.

## Menu Items

| Command | Action |
|---------|--------|
| **Lab Homepage** | Opens the NukeLab home page in a new tab |
| **Log Out** | Logs out of the current NukeLab session |
